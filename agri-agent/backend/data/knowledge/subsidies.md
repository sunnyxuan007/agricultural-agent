# 农业补贴政策

## 耕地地力保护补贴
- **对象**：拥有耕地承包权的种地农民  
- **标准**：约120元/亩  
- **条件**：不改变耕地用途、不抛荒  

## 农机购置与应用补贴
- **对象**：从事农业生产的农民和农业生产经营组织  
- **方式**：自主购机、定额补贴、先购后补  
- **范围**：高性能播种机、智能高速插秧机、大型联合收获机械等  

## 小麦“一喷三防”补助
- **对象**：承担实施任务的主体  
- **方向**：小麦中后期防虫、防病、防倒伏/干热风  

## 稻谷补贴
- **对象**：水稻种植户  
- **标准**：50-100元/亩（各省自定）  
- **申报截止**：通常在每年7-8月  

## 农业保险保费补贴
- **对象**：种植、养殖、森林保险投保主体  
- **范围**：三大粮食作物及大豆完全成本保险和种植收入保险全国实施
