# 作物种植技术

## 玉米（黄淮海夏玉米）
- **关键农时**：中南部6月15日前、北部6月20日前完成播种  
- **建议**：亩保苗4500株，小喇叭口期前后追肥  
- **风险**：连阴雨、倒伏、干旱、高温热害  

## 小麦（冬小麦）
- **关键农时**：穗期至扬花期重点防控  
- **建议**：赤霉病坚持预防为主，一喷三防  
- **风险**：赤霉病、蚜虫、干热风、烂场雨  

## 水稻
- **关键农时**：黄熟期及时收获  
- **建议**：90%以上转黄即可机械收获，遇连阴雨抢收  
- **风险**：连续降雨、倒伏、穗上发芽  

## 设施草莓
- **要点**：基肥亩施有机肥2-3方，重视腐熟有机肥，避免盐害  
- **风险**：灰霉病、白粉病、根腐、低温寡照
