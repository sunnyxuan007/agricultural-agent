# 涉农贷款产品

## 惠农e贷
- **机构**：中国农业银行  
- **额度**：3000元～50万元  
- **期限**：最长8年  
- **利率**：3.30%～4.60%  
- **担保方式**：信用、抵押、保证  
- **办理渠道**：农行掌银、网点  

## 农担e贷
- **机构**：中国工商银行  
- **额度**：最高500万元  
- **期限**：最长3年  
- **利率**：3.20%～4.30%  
- **担保**：政策性担保增信  
- **办理渠道**：工行网银、手机银行、兴农通App  

## 农户e贷
- **机构**：中国工商银行  
- **额度**：信用最高30万元，增信最高50万元  
- **利率**：3.40%～4.80%  
- **担保**：信用/增信  
- **办理渠道**：区域合作场景、工行渠道  

## 农牧贷
- **机构**：中国邮政储蓄银行  
- **额度**：未公开（一般5-50万元）  
- **利率**：3.60%～5.20%  
- **担保**：免担保、免抵押、纯信用  
- **办理渠道**：邮储银行网点  

## 农业小企业贷款
- **机构**：中国农业发展银行  
- **额度**：单户授信总额3000万元以下  
- **期限**：中期流动资金贷款  
- **利率**：3.00%～4.20%  
- **办理渠道**：农发行分支机构
